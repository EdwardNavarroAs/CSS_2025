Modelo de Caja en CSS (Box Model)

Aprende cómo los navegadores calculan el espacio que ocupa cada elemento.

1. ¿Qué es el Modelo de Caja?

En CSS, cada elemento se representa como una caja rectangular. El modelo de caja está compuesto por:

  • Contenido - El área donde aparece el texto o imagen.
  • Relleno (padding) - Espacio entre el contenido y el borde.
  • Borde (border) - Contorno del elemento.
  • Margen (margin) - Espacio entre el elemento y otros elementos.

3. box-sizing

Por defecto, el tamaño (width y height) se aplica solo al contenido. Con box-sizing: border-box, el tamaño incluye padding y borde, lo cual facilita el control del diseño.
