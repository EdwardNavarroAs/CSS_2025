10 - Flexbox

Flexbox (Flexible Box Layout) es una técnica de diseño en CSS que nos permite alinear y distribuir espacio entre elementos de manera eficiente, especialmente cuando el tamaño de los elementos es dinámico o desconocido. Funciona en una sola dimensión: podemos organizar elementos en filas (row) o en columnas (column), pero no en ambas al mismo tiempo.

📐 Dirección de los elementos

Una de las primeras propiedades que usamos con Flexbox es flex-direction, que define si los elementos se colocarán en fila (horizontal) o en columna (vertical).

➡️ Dirección: fila (row)

Esta es la dirección por defecto. Los elementos hijos se colocan uno al lado del otro, de izquierda a derecha.

📏 flex-grow, flex-shrink y flex-basis (control granular)

Estos tres valores controlan cómo se comportan los ítems dentro de un contenedor flexible:

  • flex-grow: qué tanto puede crecer el ítem cuando hay espacio disponible.
  • flex-shrink: qué tanto puede encogerse el ítem cuando no hay suficiente espacio.
  • flex-basis: el tamaño inicial preferido del ítem antes de aplicar grow o shrink.

En el siguiente ejemplo, el primer ítem puede crecer 1 parte, el segundo 2 partes, y el tercero tiene un tamaño base de 100px pero no crecerá:
