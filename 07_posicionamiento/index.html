Posicionamiento en CSS

Aprende a ubicar elementos en la página usando diferentes técnicas.

1. position: static (por defecto)

Caja con posición estática

2. position: relative

Caja relativa (desplazada desde su posición original)

3. position: absolute

Caja absoluta (se posiciona respecto al contenedor)

4. position: fixed

Caja fija (permanece en la pantalla al hacer scroll)

5. position: sticky

Este párrafo es sticky. Se queda "pegado" al llegar al borde superior al hacer scroll.
