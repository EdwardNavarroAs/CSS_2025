Curso de CSS: Selectores

Los selectores son las herramientas que usamos para indicar a qué elementos HTML queremos aplicar un estilo. Existen distintos tipos, desde los más simples como etiquetas, hasta avanzados como atributos o combinadores.

1. Selector de etiqueta

Este párrafo será estilizado usando un selector de tipo etiqueta (por ejemplo, p). Este tipo de selector aplica el mismo estilo a todos los elementos del mismo tipo.

2. Selector de clase

Este párrafo usa una clase llamada texto. Las clases se usan para aplicar estilos a múltiples elementos que comparten una misma categoría.

3. Selector de ID

Este párrafo tiene un ID único (parrafo-id). Los ID se usan para identificar y dar estilo a un único elemento en la página.

4. Selectores de atributos

Podemos aplicar estilos a elementos que tengan ciertos atributos, como href, src o type.

5. Selectores por relaciones entre elementos

Estos selectores permiten aplicar estilos según la posición de un elemento respecto a otro (por ejemplo, hijos directos, hermanos adyacentes o descendientes).

6. Selectores anidados (CSS moderno)

En CSS moderno, especialmente con preprocesadores o soporte actual, podemos anidar reglas dentro de otras para construir selectores más complejos de forma más legible.
