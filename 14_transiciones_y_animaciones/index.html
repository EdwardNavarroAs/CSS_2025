🎬 Transiciones y Animaciones en CSS

📘 ¿Qué es una transición?

Una transición permite que los cambios en una propiedad CSS ocurran de forma gradual. Por ejemplo, si cambiamos el color o el tamaño de un botón al pasar el cursor, una transición hace que ese cambio sea suave en vez de inmediato.

📘 ¿Qué es una animación?

Una animación define una serie de cambios que ocurren en el tiempo, usando reglas llamadas @keyframes. A diferencia de las transiciones, las animaciones no requieren interacción del usuario y pueden repetirse automáticamente.
