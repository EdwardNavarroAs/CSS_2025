03 - Pseudoclases y Pseudoelementos

Explora cómo aplicar estilos a estados específicos o partes internas de un elemento usando pseudoclases y pseudoelementos.

¿Qué son las pseudoclases?

Las pseudoclases son palabras clave precedidas por dos puntos (:) que se agregan a un selector para definir un estado especial del elemento. Por ejemplo, cuando el usuario pasa el mouse por encima, cuando un campo está enfocado, cuando un enlace fue visitado, etc.

  • Enlace no visitado
  • Enlace visitado (simulado)

¿Qué son los pseudoelementos?

Los pseudoelementos permiten aplicar estilos a una parte específica de un elemento, como la primera letra, la primera línea o para insertar contenido antes o después del texto. Se escriben con dos puntos dobles (::).
