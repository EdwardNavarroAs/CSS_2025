01 - Introducción a CSS

CSS (Cascading Style Sheets) se usa para definir el estilo visual de una página HTML.

¿Qué es CSS?

CSS permite aplicar colores, tipografía, tamaños, espaciados, y posicionamiento a los elementos HTML.

¿Cómo se enlaza CSS a un documento HTML?

Usamos la etiqueta <link> dentro del <head> del HTML para enlazar un archivo CSS externo.

<link rel="stylesheet" href="styles.css">

Sintaxis básica de CSS

CSS sigue esta estructura:

                
                selector {
                    propiedad: valor;
                }

Ejemplo:

                h1 {color: steelblue;}

Relación entre HTML, CSS y el DOM

El DOM (Document Object Model) es una representación estructurada del documento HTML. CSS interactúa con el DOM para aplicar estilos visuales a los elementos.

Por ejemplo, si tenemos un elemento <p> en el HTML, CSS puede cambiar su color, tamaño o posición.

Importante sobre compatibilidad

Si un navegador no entiende una propiedad CSS, simplemente la ignora y aplica las siguientes que sí reconoce. Esto permite usar CSS moderno sin romper el diseño.

p {
            colour: blue; /* Error: mal escrito, se ignora */
                font-size: 18px; /* Se aplica */
            }
